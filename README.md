# 项目简介

一个基于网易云API的 vue 项目（样式也是仿网易云）

api来源：https://github.com/Binaryify/NeteaseCloudMusicApi





# Vue 3 + TypeScript + Vite





# 运行

```bash
npm install
npm run dev
```
