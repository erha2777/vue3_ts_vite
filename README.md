# 项目简介

一个的仿网易云音乐播放器

api来源：https://github.com/Binaryify/NeteaseCloudMusicApi



# 运行

```bash
npm install
npm run dev
```

# 功能

**网易云扫码登录（挺慢的）**

**用户歌单**

**私人FM**

**搜索歌曲**

**每日歌曲推荐**

**右键收藏到歌单**

**歌曲播放（如果有问题需要将网站设置的声音权限从自动改成允许）**

**歌曲信息**

**主题切换**
